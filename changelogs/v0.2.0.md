# 0.2.0-beta (14.05.2026)
## ➕ Новое
➕ Красивое форматирование чисел. 123456789 -> 123.456.789\
➕ Команда `ферма`. Добывает ирис-коины **от 5 до 50**. В зависимости от количества времени
которое не используется команда множитель будет расти от 1.0 до 3.0 (до сотых -> 1.23)\
➕ Команда `топ <число>` - выводит глобальный топ пользователей по ирис-коинам. Да, не особо
верно сделано, в будущем будет сделано примерно так: `гтоп ирис-коины <число>`\
По дефолту выводит **топ 30** пользователей\
➕ Теперь к глобальному профилю прилагается онлайн ли участник и ссылка на его профиль в его имени
## 🍳 Изменено
❗🏷️ Колонка `about_me` -> `motto`. Требуется переименование столбца в таблице `bank`\
🏷️ Команда `о себе` -> `девиз`
